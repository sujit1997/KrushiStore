This GitHub repository contains an automation testing project for the Krushi Store website. The project is developed using Java, Selenium WebDriver, and TestNG, providing a robust and scalable framework for testing e-commerce platforms focused on agricultural products.

_**Project Features**_

_**Comprehensive Test Suite:**_ Automates the key functionalities of the Krushi Store website, ensuring a seamless user experience.

_**Selenium WebDriver:**_ Enables interaction with web elements to simulate real user actions like browsing, purchasing, and account management.

_**TestNG Framework:**_ Supports annotations, test execution control, and detailed reporting.

_**Page Object Model (POM):**_ Implements a modular and maintainable design for test scripts.

_**Cross-Browser Testing:**_ Ensures website compatibility and functionality across various browsers.

_**Data-Driven Testing:**_ Allows execution of test cases with multiple datasets using external sources like Excel or JSON.

_**Custom Reporting:**_ Provides detailed reports of test case execution for analysis and debugging.

**Key Functionalities Tested**

_**User Registration and Login:**_ Validation of sign-up, login, and logout functionalities.

_**Product Search and Categories:**_ Ensures accurate product search results and correct categorization.

_**Add to Cart:**_ Tests adding, updating, and removing products from the cart.

_**Checkout Process:**_ Validates the entire order placement workflow, including address selection and payment integration.

_**Order Management:**_ Verifies order history and status updates.

_**Responsive Design:**_ Ensures the website works well on different devices and screen sizes.

_**Error Handling:**_ Captures screenshots and logs for failed test cases to assist in debugging.

**Requirements**
